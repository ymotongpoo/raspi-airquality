# raspi-airquality
Air quality polling script with BME680

## Prerequisites
### Hardware
* Raspberry Pi
* [BME680](https://shop.pimoroni.com/products/bme680-breakout)

### Software
* Python 3.5+
* `i2c-tools`, `read-edid`, `libi2c-dev` (In the case of Raspbian 9 Stretch)
